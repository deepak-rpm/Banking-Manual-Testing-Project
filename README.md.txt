# Banking Manual Testing Project

## Project Overview
This project contains manual testing documentation for the Guru99 Banking application based on the SRS document.

## Domain
Banking / Net Banking

## Testing Type
- Functional Testing
- UI Testing
- Validation Testing
- Positive Testing
- Negative Testing
- Regression Testing

- Login
- Logout
- New Customer
- Edit Customer
- Delete Customer
- New Account
- Edit Account
- Delete Account
- Deposit
- Withdrawal
- Fund Transfer
- Balance Enquiry

- Customized Statement
- Change Password

## Documents Included
- Software Requirements Specification
- Test Cases
- RTM
- Bug Report Template
- Coverage Summary

## Tools Used
- MS Excel
- Manual Testing Concepts
- Requirement Traceability Matrix
- Defect Reporting
